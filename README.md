# DevRantWebApp
Unofficial WebApp for devRant built with Angular 2, TypeScript &amp; MDL

- requirements
  - node
  - npm
  - angular-cli

- install 
```bash
npm install
````

- run 
```bash
ng serve
open http://localhost:4200
```
